# vue i18n query

> 通过接口查询后台的多语言配置，匹配Vue文件中的中文，在工具栏中展示匹配到的内容。并可以做替换为对用i18n code的操作。可以做到一键替换的效果。提高开发替换多语言效率。
## 已完成项

- [x] 支持从接口获取多语言内容，并匹配到当前激活vue文件中的中文
- [x] 支持将匹配到当前文件的语言展示到左侧view中
- [x] 支持{0}这种组件插值
- [x] 添加配置选项
- [x] 打包本地插件
- [x] 安装
- [x] 多语言展示改为树结构
- [x] 没查找到的中文不做替换
